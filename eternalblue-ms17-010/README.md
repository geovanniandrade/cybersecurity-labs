# EternalBlue (MS17-010) Exploitation Lab

## Objective

Demonstrate exploitation of the SMB vulnerability MS17-010 (EternalBlue) in a controlled lab environment.

## Vulnerability

MS17-010 is a critical vulnerability in Microsoft's SMB protocol that allows remote code execution on vulnerable Windows systems.

This vulnerability was widely exploited by attacks such as WannaCry ransomware.

## Tools Used

- Kali Linux
- Nmap
- Metasploit Framework

## Steps Performed

1. Network reconnaissance using Nmap
2. Identification of open SMB ports
3. Verification of MS17-010 vulnerability
4. Exploitation using Metasploit
5. Gaining remote access to the target machine
